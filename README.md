# 📄 PDF页面提取工具 - 网页版

一个基于Streamlit的Web应用程序，用于从PDF文件中提取指定页面并保存为新的PDF文件。

## ✨ 核心特性

- 🖥️ **现代化Web界面**：无需命令行，所有操作在浏览器中直观完成
- 📑 **智能页面选择**：支持单个、多个、连续范围及奇偶页等多种选择模式
- 🚀 **批量处理**：一键上传并处理多个PDF文件，效率倍增
- 👁️ **实时预览**：提取前可预览页面内容，确保选择无误
- 🐳 **多方式部署**：支持本地运行、Docker容器及云平台部署
- 🔒 **安全处理**：文件仅在内存和临时目录处理，自动清理无残留

## 🌟 功能特性

1. **单文件提取**：上传单个PDF，选择特定页面提取
2. **批量处理**：同时处理多个PDF文件，每个文件可设置不同的提取规则
3. **多种选择方式**：
   - 手动输入页码（支持逗号分隔和连字符范围）
   - 范围选择
   - 奇数/偶数页选择
4. **页面预览**：查看PDF内容预览，辅助选择页面
5. **实时反馈**：显示处理进度和结果统计
6. **一键下载**：提取完成后可直接下载新PDF文件

## 部署方式

### 方法一：本地运行（推荐开发使用）

```bash
# 1. 克隆仓库
git clone https://github.com/liuqingrong/PDF_split.git
cd PDF_split

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动应用
streamlit run app.py

### 方法二：Docker 部署（推荐生产环境）

```bash
# 1. 构建Docker镜像
docker build -t PDF-split .

# 2. 运行容器
docker run -p 8501:8501 PDF-split

# 3. 访问应用
打开浏览器并访问 http://localhost:8501

### 方法三：一键脚本运行

```bash
# Linux/Mac
chmod +x run.py
./run.py

# Windows
python run.py

# 📖 详细使用指南
1. 单文件提取模式

2. 在侧边栏导航点击"提取页面"

3. 点击上传区域或拖拽PDF文件至指定区域

4. 系统自动分析PDF并显示总页数和前10页预览

5. 选择页面选择方式：
**手动输入**：使用灵活语法指定页面
**范围选择器**：通过滑块选择起始和结束页
**奇偶过滤**：一键选择所有奇数页或偶数页
6. 点击"开始提取页面"按钮
7. 处理完成后，点击下载按钮获取新PDF
